# 快速开始指南

## 🚀 5分钟上手

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问: http://localhost:3004

### 3. 开始使用

现在你可以：

#### ✅ 查看现有任务
- 浏览器会自动加载 37 个初始任务
- 11 个模块，包含 H1 和 H2 两个阶段

#### ✅ 添加新任务
1. 点击工具栏的 "添加任务" 按钮
2. 填写任务信息：
   - 任务名称
   - 任务类型（模块/任务/子任务）
   - 日期范围
   - 负责人
   - 进度
3. 点击 "创建" 按钮

#### ✅ 编辑任务
1. 双击任何任务行
2. 在弹窗中修改属性
3. 点击 "更新" 保存

#### ✅ 拖拽调整
- 拖动任务条：调整开始时间
- 拖动任务条边缘：调整持续时间
- 拖动进度条：调整完成百分比

#### ✅ 切换视图
点击工具栏的视图按钮：
- **日**: 详细的每日视图
- **周**: 每周规划
- **月**: 月度规划（默认）
- **季度**: 季度规划

#### ✅ 导出数据
点击 "导出" 按钮，下载 JSON 文件备份

#### ✅ 导入数据
点击 "导入" 按钮，上传之前导出的 JSON 文件

## 🔧 常见问题

### Q: 数据会丢失吗？
A: 数据自动保存在浏览器的 localStorage。只要不清除浏览器缓存，数据会一直保留。建议定期导出备份。

### Q: 如何修改初始数据？
A:
1. 编辑 `../gantt_chart.py` 中的 `tasks_list`
2. 运行 `python3 ../backend/data_exporter.py`
3. 重新加载页面

### Q: 如何修改配色？
A: 编辑 `src/styles/gantt-theme.css`

### Q: 支持多人协作吗？
A: 当前版本使用 localStorage，不支持多人协作。未来版本会添加后端 API。
